# Gestor de glosario IA

Glosario colaborativo en Markdown sobre conceptos de inteligencia artificial.

El objetivo del proyecto es construir y mantener una fuente de referencia clara, ordenada y fácil de actualizar sobre términos relacionados con IA, modelos de lenguaje, agentes, herramientas, prompts, búsqueda, automatización y conceptos técnicos asociados.

## Archivos principales

- `glosario_ia_inicial.md`: archivo principal del glosario y fuente de verdad.
- `guia_uso_proyecto_glosario_ia.md`: guía de uso y flujo de trabajo del proyecto.
- `CHANGELOG.md`: historial de versiones y cambios relevantes.
